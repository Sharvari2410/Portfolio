# Sharvari's Portfolio

A creative personal portfolio website built using only HTML, CSS, and JavaScript.

## Overview
This project showcases:
- Student bio and profile section
- Skills in a clean tag-based layout
- Project cards with custom visual assets
- Contact section with social links
- Responsive design for desktop and mobile

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript

## Project Structure
```text
portfolio/
  index.html
  assets/
    css/
      style.css
    js/
      script.js
    images/
      profile.jpg
      projects/
        project-1.png
        project-2.png
        project-3.png
        project-4.png
        project-5.png
        project-6.png
```

## How to Run
1. Open the project folder.
2. Launch `index.html` in your browser.

For best results, use Live Server in VS Code.

## Customization
- Profile photo: replace `assets/images/profile.jpg`
- Bio text: edit the Student Bio block in `index.html`
- Skills tags: edit the Skills section in `index.html`
- Project titles/descriptions/images: edit the Projects section in `index.html`
- Colors and layout: edit `assets/css/style.css`
- Interactions: edit `assets/js/script.js`
